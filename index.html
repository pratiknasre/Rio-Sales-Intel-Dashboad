How to Update the Dashboard
DAILY UPDATE PROCESS
Follow these steps every day after field activity ends (ideally by 7 PM)
1
Open your CRM / Reporting Tool
Log in to the RIO sales reporting system. Navigate to Reports → Export
2
Export: TC PC Report
Select the 1 - TC PC report sheet for today's date. This contains: Revenue, TC, PC, availability, SKU cases per rep.

Sheet columns needed: Emp ID, Name, Territory, MTD Value, Daily Value, TC, SC, PC, CAN/BOOM/GO cases, Login time, Logout time
3
Export: Timeline Report
Select the Timeline Report sheet for today's date. This contains hour-by-hour call distribution (9 AM – 8 PM) per rep.

Required for: Hourly call heatmap in Team Leader view
4
Export: Sales Team Dashboard
Select the Sales Team Dashboard sheet. This is the authoritative revenue source — all MTD totals are pulled from Row 7 of this sheet.

Critical: Always use this for final revenue numbers, not the TC PC report
5
Save as a single Excel file
Combine all 3 sheets into one .xlsx file. Name it clearly:
RIO_Sales_YYYY-MM-DD.xlsx

e.g. RIO_Sales_2026-03-01.xlsx
6
Upload to Dashboard
Click "Update Daily Data" in the top-right of the dashboard. Choose "Upload Excel" tab, drag your file in, and click Process.

The new date will appear in the calendar picker automatically.

⚠️ COMMON MISTAKES
❌ Using TC PC report for MTD revenue (use Sales Team Dashboard)
❌ Uploading without Timeline Report (breaks hourly chart)
❌ Wrong date in filename
MONTHLY PROCESS
At the start of each new month
1
Export full month history
Export each working day of the previous month individually. Upload each date one by one using the calendar picker to verify each date is loaded correctly.
2
Update SC Targets
If monthly targets change, include the updated SC Target sheet in your upload. This updates the target vs actual comparisons across all views.
3
Archive & share
Save a copy of the completed dashboard HTML file at month-end as an archive. Share the current file with TLs for the new month via WhatsApp or Google Drive.
